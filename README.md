📁 File Handling System (Python)
📌 Description<br>

This is a simple File Handling System built using Python.
It allows users to perform basic file operations like creating, reading, updating, and deleting files directly from the terminal.

The project is menu-driven and easy to use, making it ideal for beginners to understand file handling concepts in Python.

🚀 Features<br>
Create a new file<br>
Read contents of a file<br>
Update file (rename, overwrite, append)<br>
Delete a file<br>
Displays all files and folders in the current directory<br>

🛠️ Technologies Used<br>
Python<br>
Built-in modules:<br>
os<br>
pathlib<br>

📋 Menu Options<br>
When you run the program, you will see:
1. Creating a File
2. Reading a File
3. Updating a File
4. Deleting a File
5. Exit

**Choose an option by entering the corresponding number.**

🔧 Functional Overview

1. Create File
Enter file name
Add content
File gets created if it doesn’t already exist

2. Read File
Displays content of selected file

3. Update File
Rename file
Overwrite content
Append content

4. Delete File
Permanently deletes the selected file

⚠️ Error Handling
Handles invalid inputs
Checks if file exists before operations
Displays error messages if something goes wrong
